# AI 亲子计划助手 · 交互原型

这个原型更贴近“AI 做助手，人来决策”的设想：

- **项目创建**：输入孩子年龄/水平、选择学科、多行目标描述
- **AI 建议任务（可编辑）**：先弹窗确认，再加入任务池
- **任务池 → 日历**：任务拖到某一天（周视图），或点击“+ 日历”默认放到今天
- **打卡/统计**：简单完成标记与按项目统计

## 本地预览
双击 `docs/index.html` 即可。

## 发布到 GitHub Pages
1. 上传整个 `docs/` 到仓库（或覆盖原有 `docs`）
2. Settings → Pages：Branch `main`，Folder `/docs` → Save
3. 访问仓库的 Pages 链接即可
